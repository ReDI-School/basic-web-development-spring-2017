Element attributes

Elements can have attributes. Inside (start) tag:

<img src="images/test.jpg">

<p class="test"> ... </p>

Multiple attributes:

<img src="images/test.jpg" alt="An image" class="test">

Exercise

Add all the tags around this text that was rendered from HTML:

Scrambled eggs

For 2 persons

Ingredients

  • 2 Eggs
  • Salt
  • Oil

Instructions

  • Heat the pan, add oil, then add the eggs.
  • Add salt
  • Scramble the eggs
  • When the eggs are ready to eat, serve them.

Solution

<!DOCTYPE html>
<html>
<body>
<h1>Scrambled eggs</h1>
<p>For 2 persons.</p>
<hr>
<h2>Ingredients</h2>
<ul>
<li>2 Eggs</li>
<li>Salt</li>
<li>Oil</li>
</ul>
<h2>Instructions</h2>
<p>
Heat the pan, add oil, then <strong>add the eggs</strong>. Add
salt. Scramble the eggs.
</p>
<p>
When the eggs are ready to eat, serve them.
</p>
</body>
</html>

URLs

A full URL has protocol, domain, path

http://mysite.com/images/picture.jpg

Here the protocol is http, domain is mysite.com, path is /images/picture.jpg

Relative URLs

If a URL is not complete, the browser uses the current context

//mysite.com/images/picture.jpg
/images/picture.jpg
images/picture.jpg

<a href="http://myothersite.com/">The link text</a>
