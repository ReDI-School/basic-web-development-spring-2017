Pseudo class :nth-child()

:nth-child(n) selector matches every element that is the nth child, regardless of type, of its parent.

nth means 1st, 2nd, 3rd, 4th and so on. n is any whole number

The pseudo class selects the sibling element in parantheses

/* Formula */ class:nth-child(number) {} /* select the second sibling in the group of selectors */ selector:nth-child(2) {} /* select the 10th sibling in the group of selectors */ selector:nth-child(10) {}

Document Flow

Document flow refers to the way the browser positions elements on the page.

In general, the browser starts at the top left of the page and starts placing elements one after another.

Float: what is this?

Float is a CSS positioning property.

In its simplest use, the float property can be used to wrap text around images.

Floated elements remain a part of the flow of the web page.

/* Formula */ .class { float: right; }

It can be left, right, none or inherit.

Clear - what is this?

The clear property is used to control the behavior of floating elements.

Elements after a floating element will flow around it. To avoid this, use the clear property.

The clear property specifies on which sides of an element floating elements are not allowed to float:

/* Formula */ .class { clear: left; }
