Iframe

An Iframe is a website within a website.

Using an Iframe you can embed into your page:

  • videos from YouTube, Vimeo, ...
  • Google, Bing Maps, ...
  • music from SoundCloud, Spotify, ...
  • tweets and feeds from Twitter, Facebook, Instagram, ...
  • a pen from CodePen, jsFiddle, pastebin, ...
  • products from amazon, Zalando, ...
  • PDFs

And more...

Iframe Example: YouTube

Go to a YouTube page
Choose "Share" (Teilen) beneath the video
Choose "Embed" (Einbetten) and copy the code to your webpage

<iframe width="560" height="315" src="https://www.youtube.com/embed/8TN4II0iAck?ecver=1" frameborder="0" allowfullscreen></iframe>

CSS multiple classes

Imagine the following CSS rule:

.big.warning { color: red; font-size: 4em; }

It will only apply to elements, that have both classes .big and .warning.

Pseudo class :nth-child()

nth means 1st, 2nd, 3rd, 4th and so on. n is any whole number

The pseudo class selects the sibling element in parantheses

/* select the second sibling in the group of selectors */ selector:nth-child(2) {} /* select the 10th sibling in the group of selectors */ selector:nth-child(10) {}
